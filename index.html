Matrix Rotation

Given an n x n 2D matrix representing an image, rotate the image by 90 degrees (clockwise)

The code first transposes the matrix by swapping elements across its main diagonal. This is done using a nested loop, iterating through the upper triangle of the matrix and swapping elements with their corresponding elements across the main diagonal.After transposing, the code reverses each row of the matrix. This step effectively rotates the matrix 90 degrees clockwise
